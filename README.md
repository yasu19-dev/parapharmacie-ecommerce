# 💊 Parapharmacie en Ligne - Plateforme E-commerce

Une plateforme e-commerce full-stack développée en PHP natif pour une parapharmacie. Ce projet intègre un catalogue de produits dynamique, un système de panier sécurisé et la génération de reçus de commande au format PDF.

## 🚀 Fonctionnalités Principales

- **Catalogue Dynamique :** Affichage des produits et gestion des catégories via un fichier de données JSON.
- **Système de Panier :** Ajout, modification et suppression d'articles avec gestion des sessions utilisateurs (`$_SESSION`).
- **Processus de Commande :** Validation sécurisée des commandes et enregistrement des transactions dans la base de données.
- **Génération de PDF :** Création automatique de factures/reçus de commande téléchargeables et imprimables.
- **Interface Responsive :** Design adaptatif construit avec Bootstrap pour une navigation fluide sur mobile et bureau.
